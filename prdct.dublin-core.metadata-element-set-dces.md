## 15 core elements

- contributor
- coverage (spatial, temporal, jurisdicational)
- creator
- date
- description
- format
- identifier
- language
- publisher
- relation ("a related resource")
- rights
- source
- subject (keywords, key phrases, classification codes, or other controlled vocab)
